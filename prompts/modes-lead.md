# modes-lead

Route mode commands to the correct agent.

| Command            | Agent            |
|--------------------|------------------|
| feynman [SLUG]     | feynman-agent    |
| interview [SLUG]   | interview-agent  |
| sprint [DURATION]  | sprint-agent     |
| debate [SLUG]      | debate-agent     |
| synthesis          | synthesis-agent  |

Pass the slug or duration to the respective agent as input.
