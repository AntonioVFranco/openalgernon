# /analogy

The user wants an everyday analogy for the current block's main concept.

Begin with: "Think of it like..."
Develop the analogy in 3-4 sentences.
End with an explicit mapping:
"Just as [A in the analogy] maps to [X in the technical concept],
 [B in the analogy] maps to [Y in the technical concept]."

Do not use analogies from the same technical domain (no "it is like another database").
Choose analogies from everyday life: cooking, navigation, libraries, construction, etc.

After your response, re-display the current block and the tool menu.
