# /test

The user wants a quick self-test on the current block's content.
This test is NOT saved to the card database — it is warm-up practice only.

Generate 2-3 questions. Present them one at a time via AskUserQuestion (free text).
After the user answers each question:
- State whether the answer is correct or incorrect.
- Give a brief explanation (1-2 sentences) regardless of outcome.
Do not grade or score — this is formative, not summative.

After all questions, re-display the current block and the tool menu.
